Intro

This demo shows our approach to teaching full-stack Iot, from hardware to messaging to software.

We augment the excellent Viddy pinhole camera with a cheap light sensor that tells you how long to expose your image.

It's a workshop format similar to Nodebot workshop.

What you'll learn

Our demo workshop teaches the basics of Johnny Five to connect get an Arduino to read the current light level from a cheap light sensor. You'll then find out how to publish those readings over the internet where a web app will pick them up to display.

The video shows the final result.

Use familiar tools

Use the developer tools you know to learn something new. The workshop uses whichever development tools you're comfortable with so you can get started right away.

Arduino optional

You can get started even without the camera or even the Arduino. The workshop can run mock versions of the Arduino in software on your laptop.

When you have the kit, you can use the same code for real.
